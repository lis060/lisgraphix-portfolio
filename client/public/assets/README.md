# Public Assets

Place all portfolio images in this folder using the exact filenames below.

## Required filenames

### Profile / Hero
- `profile.jpg` — your headshot (About page, ~4:5 portrait)
- `hero.jpg` — hero section image (Home page, ~4:3 landscape)

### Portfolio project cards (6)
- `project-elegance.jpg` — ELEGANCE Barbering & Spa
- `project-lusso.jpg` — Lusso Homes Accra
- `project-suitup.jpg` — Suit Up! Fashion
- `project-listrack.jpg` — LisTrack Dashboard
- `project-church.jpg` — Church Program Flyer System
- `project-px.jpg` — PX Inventory Concept

### Case study covers (3)
- `case-elegance.jpg`
- `case-lusso.jpg`
- `case-suitup.jpg`

### Services section (5)
- `service-webdesign.jpg`
- `service-ecommerce.jpg`
- `service-graphic.jpg`
- `service-software.jpg`
- `service-social.jpg`

## Guidelines
- **Format**: JPG or WebP preferred (PNG for graphics with transparency)
- **Size**: Keep each file under 500KB for fast loading
- **Dimensions**:
  - Project cards: 800x600 (4:3)
  - Case study covers: 1200x630 (16:9)
  - Service images: 800x450 (16:9)
  - Profile: 600x750 (4:5 portrait)
  - Hero: 800x600 (4:3)
- **Optimise**: Use https://squoosh.app or https://tinypng.com to compress before uploading
